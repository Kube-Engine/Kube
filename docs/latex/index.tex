Kube is a C++ framework that aims building efficient and stable cross-\/platform application for 2D and 3D rendering.\hypertarget{index_autotoc_md14}{}\doxysection{What can I do with Kube ?}\label{index_autotoc_md14}
Kube was built with the idea of creating 3D voxel games, however, thanks to its modular architecture, the framework can be used only for either small and large 2D and 3D applications without paying the cost of what you don\textquotesingle{}t need.

\href{https://kube-engine.github.io/Kube/}{\texttt{ Here is the documentation.}}

The goal of this framework is to establish a stable and powerful work environment for video-\/game development.\hypertarget{index_autotoc_md15}{}\doxysubsubsection{Philosophy}\label{index_autotoc_md15}
The philosophy keywords of the framework are\+:
\begin{DoxyItemize}
\item Rigor\+: Each class is thinked for a long-\/term A\+PI support with latest technologies used right now and future technologies anticipated
\item Explicitness\+: the codebase is as readable and as explicit as possible
\item Data\+: Because of modern C\+PU weakness in memory IO, the best way to crank the speed up is though smart data modeling
