\doxysubsection{Detailed Description}
Kube \mbox{\hyperlink{classk_f_1_1_application}{Application}}. 

This class prepares and run at a high level of abstraction, a graphical application. As the Kube framework aims to be cross-\/platform, only a signle surface window can be instantied with it.
