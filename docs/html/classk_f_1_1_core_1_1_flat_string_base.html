Detailed Description

template<typename Type>
class kF::Core::FlatStringBase< Type >

Flat string is pointer-sized std::string alternative that is NOT NULL TERMINATED The implementation assumes that given character Type will never throw while manipulating it The implementation comes with two weakness : Because the size and capacity are stored on the heap if you wish to get the vector size and not lookup after that it is slower due to memory indirection Because the string don't have small optimization, it will perform worse than an std::string with short strings.
