Detailed Description

Manager of every command pool 'acquire' and 'takeBack' functions are the only thread safe ones.

Manager is used to acquire access to an auto command pool for a given frame: each command is one time submit If you wish to reuse command buffers, please use a ManualCommandPool instead of the CommandPoolManager
