)    PROPERTY_COPY(PropertyType, name, __VA_ARGS__)

Use these macros to generate a new property and register it to a meta type The property can have several tags used to generate an optimal getter and setter.

  • COPY: the property will have a copy setter
  • MOVE: the property will have a move setter
  • REF: the property will have a volatile getter that will not trigger the property signal on change
  • READONLY: the property will have a protected setter not registered to the meta type

PropertyTypeis the property type
nameis the name of the property
...all variadic arguments are used to initialize the property
