Detailed Description

An expression is an opaque accesser that implement execution of expressions You should call 'Construct' static member function to create a pointer to expression.

This class is a view that has ownership over itself, 'this' pointer is the beginning of the node data Once built cannot be nor moved as its pointer is where is the data

Why such an obscure pattern ? Because it removes a level of indirection when calling a slot
