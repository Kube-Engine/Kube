Detailed Description

template<typename Type>
class kF::Core::SPSCQueue< Type >

The SPSC queue is a lock-free queue that only supports a Single Producer and a Single Consumer The queue is really fast compared to other more flexible implementations because the fact that only two thread can simultaneously read / write means that less synchronization is needed for each operation. The queue supports ranged push / pop to insert multiple elements without performance impact.
