Detailed Description

Signal is used to store meta-data about a free / member / static function.

@ Author: Matthieu Moinvaziri @ Description: Meta Signal

Each signal is attached to a class. Signal aren't thread safe. However, each class is separated from others: you can use two classes' signals on two different threads
